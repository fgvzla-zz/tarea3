Como parte práctica de esta tarea, tendrás que seguir los siguientes pasos:

  1. Igual que en la tarea 1, crea un archivo html con las respuestas a las preguntas de esta tarea. Coloca este archivo en ~/public_html/tarea2/
  2. Instala nginx en tu computador
  3. Crea un archivo de configuración para un nuevo virtual host. Configura dicho virtual host para que sirva los archivos en ~/ public_html/ utilizando el hostname tarea2.talpor.com
  4. Crea una página web que muestre tus respuestas a las preguntas teóricas presentadas al final de este correo. Cada respuesta debe presentarse en un elemento separado cuyo contenido debe ser cargado usando AJAX

Preguntas

1. ¿Qué problema resuelven las herramientas virtualenv y RVM (o rbenv)? ¿Para qué lenguaje se utiliza cada una?

Las herramientas para la creación de ambientes virtuales, permiten la creación de ambientes adaptados a un cierto proyecto, todo dentro de una misma computadora. Esto permite probar proyectos con distintas liberias, versiones de frameworks, etc. sin alterar el proyecto original. En caso de Python se utiliza la herramienta virtualenv, y para Ruby RMV.

2. ¿Qué framework piensas usar?

Django.

3. ¿Qué patrón de diseño utiliza ese framework?

Utiliza el patron MTV, o , Model - Template - View. Cuando uno solicita un URL, este se envía a una vista. La vista llama al Modelo y prepara los datos de salida. Estos son enviados a una plantilla y se envía como respuesta.

4. En el framework seleccionado, ¿cómo se crea la relación entre las tablas de la base de datos y las clases definidas?

En Django se definen modelos para poder acceder a las tablas de la base de datos. Normalmente es un modelo por cada tabla que se quiere consultar, definiendo todas las consultas que se quieran hacer en la tabla relacionada para que las vistas soliciten la información necesaria.

5. ¿Para que se utilizan PIP y Bundler?

PIP y Bundler son herramientas para el manejo de packetes escritos en Python y Django respectivamente. Estos manejadores facilitan la instalación, manejo y actualizaciones de paquetes, bibliotecas, y programas que facilitan la programación en dichos lenguajes.

5. ¿Qué es un Requirements file y un Gemfile?

Dichos archivos describen una precedencia, que los paquetes que estan escritos dentro del archivo, deben tener prioridad a la hora de resolver las dependencias de un proyecto.
